Activity: Exploring CSS Frameworks

By Josh Bailey

Date: September 25, 2025

Viewport

The screen width, measured in pixels, on a web browser running in a device.

Flexbox

A CSS value that creates a flexible container out of the HTML element it is assigned to and shows content that expands, contracts, and rearranges itself depending on the screen size.

Grid

A two-dimensional layout system used to arrange content into rows and columns.
